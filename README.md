# 🎬 Drama Recommender System

A simple Machine Learning-based drama recommender system using **TF-IDF + Cosine Similarity**.

---

## 🚀 Features
- Recommend similar dramas  
- Fuzzy search (handles typos)  
- Filter by rating & genre  
- Poster fetching using TMDB API  

---

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
